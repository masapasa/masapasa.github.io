### 进入该地址下环境

```bash
cd + space + 目录地址
```

### 查看 IP

```bash
ipconfig/all
```

### 注册表管理器

```bash
regedit
```

### jdk 版本

```bash
java -version
```

### jdk 路径

```bash
java -verbose
```

### 查找关键字

```bash
set|findstr/I
```

### 查看 gcc 版本

```bash
gcc --version
```

### 数字签名

```bash
keytool -genkey -alias android.keystore -keyalg RSA -validity 20000 -keystore android.keystore
```
