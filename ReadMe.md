# [saber2pr.github.io](https://saber2pr.github.io/)

> saber2pr's blog.

```bash
git clone https://github.com/Saber2pr/saber2pr.github.io.git --depth=1

cd ./saber2pr.github.io

yarn install

yarn dev
```

# Feature

1. 前端路由

2. markdown 博客 + 树形目录

3. 响应式布局，PC 端移动端自适应

4. 更新日志生成

5. 支持 PWA 桌面离线应用

# Publish

```bash
# 编译项目
yarn build
```

```bash
# 生成更新日志
yarn update
```
